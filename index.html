Plex for Alexa

Alexa Skill for interacting with your Plex Media Server and Plex players.

Download .zip Download .tar.gz View on GitHub

Usage

"Alexa, ask the home theater what's on deck?"

"Alexa, tell the home theater to put on Always Sunny"

"Alexa, tell the home theater to put on episode 3 of Band of Brothers"

"Alexa, tell the home theater to put on Adventure Time, and make it a good one!"

Limitations

Currently, Plex for Alexa does not have the ability to customize which server and which player you are using. It works best if you have a single Plex server connected to a single player (servers shared with you are ignored currently, so you don't have to worry about them.)

The ability to use multiple servers at the same time as well as casting to any number of players is currently under development.

Compatibility

Right now playback commands have only been tested with the following clients:

  • Plex Home Theater

If you have other players and would like to help out with testing, please join the Gitter Chatroom
